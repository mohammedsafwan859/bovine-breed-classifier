About This
Project

A final year ML capstone project dedicated to preserving and identifying India's rich bovine heritage. Using ResNet-50 deep learning, we bridge the gap between traditional agriculture and modern data science — providing accurate classification for 41 distinct breeds.

database

The Dataset

A curated collection of images representing 41 bovine breeds, sourced from the Kaggle dataset lukex9442/indian-bovine-breeds. Includes both indigenous Indian breeds and exotic breeds.

memory

The Model

ResNet-50 architecture with transfer learning via fastai + PyTorch. Trained for 20 epochs with a learning rate of 0.0012. Exported as a fastai .pkl learner for fast inference.

layers

The Stack

Flask backend · Groq LLM API (Llama 3.1) for breed insights · Vanilla JS frontend · ResNet-50 for image classification.
